# Docs Layout

- `CONTEXT.md` - shared language and durable project context.
- `docs/prd/` - product requirements documents.
- `docs/issues/` - local implementation tickets.
- `docs/adr/` - architecture decision records.
- `docs/architecture/` - current architecture notes.
- `docs/conventions/` - coding and module conventions.
- `docs/agents/README.md` - cross-assistant guide for using repo agent workflows and skills.
- `docs/agents/skills/` - vendored agent skill library for Codex, Claude, Cursor, Windsurf, Antigravity, Gemini, Copilot, and other assistants.
- `docs/agents/skills/ui-ux-pro-max/` - vendored UI/UX design intelligence skill with searchable design data, stack guidance, and Claude-style skill files.
