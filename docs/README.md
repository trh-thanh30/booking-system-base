# Tài Liệu Dự Án

Đây là điểm bắt đầu cho tài liệu kỹ thuật của `booking-system-base`.

## Bắt Đầu

- `docs/getting-started.md`: cài đặt, cấu hình môi trường và chạy repo lần đầu.
- `docs/development.md`: workflow phát triển hằng ngày.
- `docs/env.md`: quy ước biến môi trường.
- `docs/testing.md`: chiến lược test và lệnh kiểm tra.
- `docs/deployment.md`: build, Docker Compose và CI/CD.

## Kiến Trúc

- `docs/architecture/overview.md`: tổng quan kiến trúc monorepo.
- `docs/architecture/frontend-folder-structure.md`: cấu trúc client Next.js.
- `docs/architecture/frontend-route-template.md`: template tạo route frontend mới.
- `docs/architecture/frontend-shared-hooks.md`: bộ React hooks dùng chung.
- `docs/architecture/backend-folder-structure.md`: cấu trúc backend theo Clean Architecture.

## Quy Ước

- `docs/conventions/modules.md`: quy ước module nghiệp vụ.
- `docs/adr/`: quyết định kiến trúc đã chốt.
- `docs/prd/`: tài liệu yêu cầu sản phẩm.
- `docs/issues/`: issue/ticket triển khai nội bộ.

## Tích Hợp

- `docs/integrations/telegram.md`: thông báo CI/CD qua Telegram.
- `docs/agents/README.md`: hướng dẫn dùng agent skills trong repo.
